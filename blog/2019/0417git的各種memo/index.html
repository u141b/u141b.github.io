Git的各種memo

  • 2019-04-17
  • Tags: Git

給自己memo用的快查清單。（隨時更新）

VSCcode

  • 更改預設終端機
    1. 開啟設定的json
    2. 搜尋 terminal.integrated.shell.windows
    3. 更改路徑 ..\\Git\\bin\\bash.exe

Git Bash

（指令同Linux）

  • 更改起始位子
    開啟Git\etc\bash.bashrc，在文末加上cd 路徑。

  • ID設定
    設定名稱、信箱
    $ git config --global user.name "name"
    $ git config --global user.email "mail"
    顯示設定清單
    $ git config --global --list
