Markdown常用語法

  • 2019-04-09
  • Tags: Markdown

架了Hugo來玩。
Hugo沒有後台編輯區，要PO文是用Markdown編寫內容。
這裡memo幾個使用率高的語法。（先展示後放語法。）

段落

以空白行分隔。

斷行

行末兩個空格。

標題

h1

h1  or # h1
===

h2

## h2

以下列推。

字體

斜體 : *斜體* = _斜體_
粗體 : **粗體** = __粗體__
可以 組合 : _可以 **組合**_
刪除線 : ~~刪除線~~

連結

  • http://github.com : 自動生成
  • GitHub : [GitHub](http://github.com "github")

不過Markdown沒有另開視窗等功能，所以有需要的話還是要用Html寫。

圖片

![圖片內容](圖片網址 "圖片標題")

清單

  • Item 1
  • Item 2
    • Item 2a
    • Item 2b
- Item 1
- Item 2
  - Item 2a
  - Item 2b
  1. Item 1
  2. Item 2
  3. Item 3
    1. Item 3a
    2. Item 3b
1. Item 1
2. Item 2
3. Item 3
   1. Item 3a
   1. Item 3b (編號照順序或沒照順序都可以)

引用

引用的範例：
> 引用的內容
>> 還可以來個兩層

> 引用的內容
>> 還可以來個兩層

水平線

---、***、___都可以，最少三個。

行內語法

插入<br />語法 。

插入`<br />`語法 。

語法演示

<p>列表：</p>
<ul>
<li>A1</li>
<li>A2</li>
</ul>

‵ ‵ ‵ html (語法種類)
<p>列表：</p>
<ul>
<li>A1</li>
<li>A2</li>
</ul>
‵ ‵ ‵ (因為編碼會出錯所以這裡打全形，實際上 ‵ 都是 ` )

表格

A B C
1: :2: :3 
| A | B | C |
|---|---|---|
| 1:|:2:|:3 | 
(其實沒空格也可以，只是我覺得這樣比較整齊)

表格的部分還有其他進階用法，但疑似要再加掛才有效果。
目前這些就蠻夠用的，還可以自由搭配再變化。
要注意空白跟斷行。
